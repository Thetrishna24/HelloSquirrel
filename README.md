# HelloSquirrel

A simple Android “profile card” demo app—displays a name, bio and hobbies.

---

## 🚀 Features

- 📝 Shows your full name (pulled from string resources)  
- 💬 Bio section under your name  
- 🎾 Hobbies list

## 🛠️ Getting Started

### Prerequisites

- Android Studio Flamingo or later  
- Android SDK 31+  
- Java 11
